# 🔮 TaroTaro

당신의 내면을 비추는 거울 - AI 기반 타로 카드 리딩 서비스

## ✨ 주요 기능

- **다양한 스프레드**: 원카드, 쓰리카드, 켈틱크로스
- **AI 해석**: OpenAI를 활용한 개인화된 타로 해석
- **다국어 지원**: 한국어, 영어, 일본어
- **반응형 디자인**: 모바일과 데스크탑 모두 최적화
- **결과 공유**: 이미지로 결과를 저장하고 공유

## 🚀 기술 스택

- **Frontend**: React 19, Vite, Framer Motion
- **Backend**: Vercel Serverless Functions
- **AI**: OpenAI GPT API
- **Deployment**: Vercel

## 📦 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```
